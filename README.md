#  SauceDemo Automation & Testing Framework

A comprehensive test automation project built for the SauceDemo application, focusing on validating core e-commerce functionalities using both manual and automated testing approaches.

---

## Project Overview

This project covers the end-to-end user journey of the SauceDemo application:
Login → Product Listing → Add to Cart → Checkout → Order Confirmation

The objective is to ensure functional correctness, UI behavior, and data consistency across the application.

---

##  Tech Stack

- Java  
- Selenium WebDriver  
- Cucumber (BDD)  
- TestNG  
- Maven  
- Page Object Model (POM)  

---

##  Framework Features

-  Behavior Driven Development (Cucumber)  
-  Page Object Model for maintainability  
-  Reusable utility methods  
-  End-to-End automation coverage  
-  TestNG integration  
-  HTML & Extent reporting  
-  Scalable and modular framework design
